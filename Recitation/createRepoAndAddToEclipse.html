1. Once you have signed onto GitHub, click on the "Create new" icon in the top right of the screen
  2. Select "new repository"
  3. Title the respository "CMU15-313Rec1"
  4. Make a folder to hold this project on your computer and then navigate to the folder
  5. Follow the instructions to create new repository from the command line.
  6. Import the git project into your IDE if the IDE supports importing git projects. For Eclipse, the process is as follows (File >> Import >> Git >> Projects From Git >> Next >> Existing Local Repository >> Next >> Add >> check the folder you made >> Next >> Import using the new project wizard >> Java Project >> Next >> name the project >> uncheck Use default location >> Browse >> select the git folder >> Finish)
